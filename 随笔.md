## note

### jsdom
> jsdom 是一个库，用于在单元测试时，模拟 dom 环境，从而将被测试的元素渲染出来，方便对 UI 进行测试

vitest 支持使用`jsdom`或 `happy-dom`这两个库来模拟 dom 环境


### testing-library-react

**Vitest 基于 Testing-Library 提供了测试、断言等 API 、jsdom 提供了模拟 dom 环境的能力**
